      Get Started

      Fast and Full-featured

      Simple and easy to use, yet powerful and feature-rich.

      Markdown is All You Need

      Compose with just Markdown. Enrich with Shortcode components.

      Full Text Search

      Built-in full text search with FlexSearch, no extra setup required.

      Lightweight as a Feather

      No dependency or Node.js is needed to use Hextra. Powered by Hugo, one of the fastest static site generators, building your site in just seconds with a single binary.

      Reponsive with Dark Mode Included

      Looks great on different screen sizes. Built-in dark mode support, with auto-switching based on user’s system preference.

      Build and Host for Free

      Build with GitHub Actions, and host for free on GitHub Pages. Alternatively it can be hosted on any static hosting service.

      Multi-Language Made Easy

      Create multi-language pages by just adding locales suffix to the Markdown file. Adding i18n support to your site is intuitive.
